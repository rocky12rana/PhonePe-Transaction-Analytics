# 📊 PhonePe Transaction Analytics Dashboard - Power BI

## Dashboard Overview

This interactive Power BI dashboard provides an executive-level analysis of PhonePe transaction performance during 2024.

The dashboard focuses on monitoring:

- Overall transaction performance
- Customer activity
- Transaction growth trends
- Service contribution
- Payment failures
- Business performance indicators

The dashboard was designed to help stakeholders quickly understand operational performance and identify improvement opportunities.

---

# Dashboard Objectives

The main objectives of this dashboard are:

- Track overall transaction KPIs
- Analyze monthly transaction trends
- Identify growth patterns
- Compare service performance
- Monitor transaction failures
- Understand customer engagement

---

# Key Dashboard Features

## 1. Executive KPI Summary

The dashboard provides high-level performance indicators:

| KPI | Purpose |
|---|---|
| Total Transaction Value | Measures overall monetary transaction volume |
| Total Transactions | Tracks transaction activity |
| Total Users | Measures customer base size |
| Success Rate | Evaluates payment reliability |
| Failure Rate | Identifies operational issues |
| Failure Cost | Measures financial impact of failed transactions |
| Transactions Per User | Measures customer engagement |
| Value Per User | Measures customer value contribution |
